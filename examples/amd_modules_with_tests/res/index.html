This is an example for a typescript project in a browser, using require.js and amd modules.

execute 

npm install
to install all dependencies

Currently this project uses the non released version 1.5 of typescript, because the compiler of ArcticTypescript uses tsconfig.json. The most recent typescript version 1.4 is not compatible with tsconfig.json.

package.json: git+ssh://git@github.com:Microsoft/TypeScript.git

A test system is included.
